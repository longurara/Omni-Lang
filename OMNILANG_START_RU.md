# Руководство пользователя OmniLang
*(Для программистов и начинающих)*

Добро пожаловать в **OmniLang**! Это простой, читаемый, но мощный язык программирования. Этот документ поможет вам изучить OmniLang с нуля.

## 1. Установка и запуск

OmniLang работает очень просто. Вам нужен только файл `omni.exe` и файл исходного кода с расширением `.omni`.

**Как запустить:**
Откройте CMD или PowerShell и введите:
```bash
omni.exe vash_fail.omni
```

**Совет:** Для подсветки синтаксиса и автодополнения кода установите расширение `vscode-omni` для Visual Studio Code.

## 2. Основные понятия

### 2.1. Переменные (Variables)
В OmniLang не нужно объявлять типы данных. Просто присвойте значение.
```omni
x = 10              # Целое число (Int)
name = "OmniLang"   # Строка (String)
score = 9.5         # Дробное число (Double)
isOk = true         # Логическое значение (Bool)
list = [1, 2]       # Список (Array)
```

### 2.2. Комментарии (Comments)
Используйте символ `#` для заметок (компьютер игнорирует этот текст).
```omni
# Это комментарий
print("Hi") # Комментарий в конце строки
```

### 2.3. Ввод и Вывод
Взаимодействие через консоль.

*   `print(...)`: Вывод на экран.
*   `input(...)`: Ввод от пользователя.

```omni
name = input("Введите ваше имя: ")
print("Привет, ", name)
```

## 3. Управление потоком (Logic)

### 3.1. Если... То... (If - Elif - Else)
Обратите внимание: **отступы обязательны** (используйте Tab).
```omni
score = 8

if score >= 9:
    print("Отлично!")
elif score >= 7:
    print("Хорошо.")
else:
    print("Нужно постараться.")
```

### 3.2. Циклы (Loops)

**Цикл `while` (Пока условие истинно):**
```omni
i = 0
while i < 5:
    print("Номер:", i)
    i = i + 1
```

**Цикл `for` (Повторить N раз):**
```omni
# От 0 до 4
for i in range(5):
    print("Число:", i)
```

## 4. Функции (Functions)
Группировка кода для повторного использования.
```omni
# Определение функции
def CalcSum(a, b):
    res = a + b
    return res

# Использование
x = CalcSum(5, 10)
print(x) # Результат: 15
```

## 5. Стандартная библиотека (Standard Library)
OmniLang имеет богатую встроенную библиотеку.

### 5.1. Математика (Math)

| Функция | Описание | Пример |
|---------|----------|--------|
| `Math.abs(x)` | Модуль числа. | `Math.abs(-5)` -> `5` |
| `Math.sqrt(x)` | Квадратный корень. | `Math.sqrt(9)` -> `3` |
| `Math.pow(a,b)` | Возведение в степень. | `Math.pow(2, 3)` -> `8` |
| `Math.round(x)` | Округление. | `Math.round(4.6)` -> `5` |
| `Math.random()` | Случайное число (0-1). | `r = Math.random()` |

### 5.2. Строки (String)

| Функция | Описание | Пример |
|---------|----------|--------|
| `len(s)` | Длина строки. | `len("abc")` -> `3` |
| `String.toUpperCase(s)` | В ВЕРХНИЙ РЕГИСТР. | `String.toUpperCase("a")` -> `"A"` |
| `String.trim(s)` | Удалить пробелы по краям. | `String.trim(" a ")` -> `"a"` |
| `String.contains(s, sub)`| Содержит ли подстроку. | `String.contains("Hi", "i")` -> `true` |
| `String.format(...)` | Форматирование. | `String.format("Hi %s", "You")` |

### 5.3. Списки (List / Array)

| Функция | Описание | Пример |
|---------|----------|--------|
| `List.new()` | Новый список. | `l = List.new()` |
| `List.add(l, v)` | Добавить элемент. | `l = List.add(l, "A")` |
| `List.get(l, i)` | Получить элемент (индекс с 0).| `x = List.get(l, 0)` |
| `List.size(l)` | Размер списка. | `n = List.size(l)` |

### 5.4. Файлы (File)

| Функция | Описание | Пример |
|---------|----------|--------|
| `File.write(path, data)` | Запись в файл. | `File.write("t.txt", "Hi")` |
| `File.read(path)` | Чтение файла. | `s = File.read("t.txt")` |
| `File.exists(path)` | Проверка существования.| `if File.exists("t.txt"):`|

### 5.5. JSON (Данные)

| Функция | Описание | Пример |
|---------|----------|--------|
| `Serializer.saveJSON(path, data)` | Сохранить в JSON. | `Serializer.saveJSON("d.json", obj)` |
| `Serializer.loadJSON(path)` | Загрузить из JSON. | `obj = Serializer.loadJSON("d.json")` |

## 6. Пример программы

### Калькулятор суммы
```omni
print("--- СУММАТОР ---")
s1 = input("Введите первое число: ")
s2 = input("Введите второе число: ")

# Преобразование строк в числа
n1 = Integer.parseInt(s1)
n2 = Integer.parseInt(s2)

res = n1 + n2
printf("Сумма %d и %d равна: %d\n", n1, n2, res)
```
